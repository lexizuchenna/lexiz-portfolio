# LexizDev Portfolio

## Overview

A high-performance personal portfolio and engineering log designed for Alexander Ukwueze. This project leverages Next.js 16 and TypeScript to provide a seamless, low-latency interface for showcasing software architecture, career milestones, and technical deployments.

## Features

- **Project Inventory:** A detailed showcase of full-stack and mobile deployments with deep-dive technical reports.
- **Career Logging:** Chronological archive of professional experience with mission overviews and key contributions.
- **Technical Stack Index:** Interactive capability visualization featuring proficiency metrics.
- **Automated Sitemaps:** SEO-optimized routing using `next-sitemap` for maximum discoverability.
- **Responsive Architecture:** Fluid UI components built with CSS Modules for modularity and performance.

## Getting Started

### Installation

1. **Clone the Repository**

   ```bash
   git clone https://github.com/lexizuchenna/lexizdev.git
   ```

2. **Install Dependencies**

   ```bash
   pnpm install
   ```

3. **Production Build**

   ```bash
   pnpm build
   ```

4. **Launch Application**
   ```bash
   pnpm start
   ```

### Environment Variables

Configure a `.env.local` file in the root directory with the following variables:

```local
SMTP_HOST=smtp.zoho.com
SMTP_USER=noreply@mail.lexiz.is-a.dev
SMTP_PASS=your_secure_password
NEXT_PUBLIC_SITE_URL=https://lexiz.is-a.dev
```

# LexizDev API

## Overview

Backend mail server implementation using Next.js Route Handlers and Nodemailer to handle secure contact form transmissions via SMTP.

## Features

- Nodemailer: SMTP Transport management
- HTML Templates: Dynamic email generation for inbound queries

## API Documentation

### Base URL

`/api`

### Endpoints

#### POST /send-mail

**Request**:

```json
{
  "name": "John Doe",
  "email": "john@example.com",
  "message": "Project inquiry regarding web development services."
}
```

**Response**:

```json
{
  "message": "Message sent",
  "status": 200
}
```

**Errors**:

- 400: Email, name and message are required
- 400: Error sending mail (SMTP failure)
- 500: Unknown error, try again
